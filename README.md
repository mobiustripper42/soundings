# soundings

LoRa wireless sensor mesh for Bay Branch Farm. Soil moisture tension (Watermark), soil temperature, and tunnel air temperature/humidity reporting to a Raspberry Pi gateway and a local time-series database.

**Target:** operational across Red Tunnel beds by March 2027, in time for the 2027 tomato transplant. Paper planning summer 2026, build September onward, shakedown in Green Tunnel through winter, Red deploy at transplant.

Scaffolded from `seeds/dev` templates. See `docs/` and `CLAUDE.md` for project configuration.
